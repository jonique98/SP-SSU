## sic/xe 머신의 gui 어셈블러 구현
sic/xe 머신의 오브젝트 파일(기계어)를 실행하여 결과를 출력하는 gui 어셈블러를 구현하였다.

오브젝트 파일은 sic/xe 머신의 instruction set을 따른다.
